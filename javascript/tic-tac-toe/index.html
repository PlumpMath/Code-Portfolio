Tic Tac Toe

This Tic Tac Toe Game is written as a jQuery plugin. It was developed by Kurt Mueller.

    Some notes:

  • Since jQuery is a prequisite, it requires a connection to the internet.
  • This project utilized a jQuery Plugin plugin pattern written by Addy Osmani
  • The A.I. algorithm utilized is the popular minimax algorithm implemented by Sarath Lakshman.
  • This build tool for this project was Grunt: The JavaScript Task Runner. Grunt comes packaged with PhantomJS, which runs javascript files through JS Hint, a Javascript Code Quality Tool, and QUnit, jQuery's official unit testing software suite, on every save.
  • While the game is playable, the initial first move is slow and the game a.i. is buggy. However, the game itself still works. The algorithm was translated from Python and there are some bugs in the code somewhere.
